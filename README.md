HR & Payroll Management System (SQL Project)

📌 Overview

A SQL-based HR and Payroll Management System designed to manage employee records, department-wise salary analysis, and payroll calculations using structured queries and real-world data analysis techniques.

---

🛠 Technologies Used

- SQL (MySQL / PostgreSQL)
- Database Design

---

📁 Project Structure

- 01_tables.sql → Table creation (Employees, Departments, Payroll, etc.)
- 02_database_setup.sql → Database creation and setup
- 03_insert_data.sql → Sample data insertion
- 04_queries.sql → SQL queries for analysis

---

🔑 Key Concepts Used

- Joins (INNER, LEFT)
- Aggregate Functions (SUM, AVG, COUNT)
- Subqueries
- GROUP BY & HAVING
- Data Filtering and Sorting

---

🚀 Project Highlights

- Normalized database structure
- Real-world HR and payroll use case
- Efficient SQL queries for data analysis
- Department-wise and employee-level insights
