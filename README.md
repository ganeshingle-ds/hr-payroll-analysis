
# 📊 HR & Payroll Management System (SQL Project)

# 🚀 Project Overview

   This project is a comprehensive SQL-based system designed to manage and analyze company data, including employee records, department structures, and payroll       calculations. It demonstrates the ability to solve real-world business problems using structured database queries.
   
# 🛠️ Technologies Used

   Database: MySQL (WorkBench)
   
   Language: SQL (Structured Query Language)
   
   Key Concepts: Data Modeling , Relational Databases, Data Analysis.

                 
# 📂 Project Structure

   The repository contains three primary SQL scripts to set up and analyze the system:
   
   01_database.sql: Script for creating the database and initial environment setup.
   
   02_tables.sql: Contains the Schema definition (Departments, Employees, Projects, Salaries, Attendance) and sample data insertion.
   
   03_queries.sql: A collection of analytical SQL queries used to derive business insights.

   
# 🔑 Key SQL Skills Applied

   I have implemented several advanced SQL techniques in this project:
   
   Joins: Used INNER JOIN and LEFT JOIN to connect data across multiple tables.
   
   Aggregations: Applied SUM, AVG, COUNT, and MAX to analyze salary distributions and attendance.
   
   Data Filtering: Efficient use of GROUP BY, HAVING, and ORDER BY for structured reporting.
   
   Complex Queries: Implementation of Subqueries and Window Functions to solve specific business logic.

   
# 💡 Business Insights (Sample Queries)

   The system provides several critical insights, such as:

   Salary Benchmarking: Identifying employees earning above the company average.

   Resource Allocation: Detecting "Idle Resources" (employees not currently assigned to any project).

   Departmental Budgeting: Calculating total payroll costs and employee headcount per department.
